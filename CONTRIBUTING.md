## How to Contribute

### Adding New Strategies

1. Fork this repository
2. Add your monetization strategy to the appropriate section
3. Include:
   - Clear description
   - Realistic price/income range
   - Difficulty level if applicable
4. Submit a pull request

### Sharing Income Reports

We love anonymized success stories! Include:
- Service/product type
- Monthly revenue range
- Time investment
- Key learnings

### Reporting Security Issues

If you find a malicious skill or security vulnerability:
1. Do NOT install or run the skill
2. Open an issue with details
3. Include the skill name and ClawHub URL
4. Describe the malicious behavior

### Guidelines

- Keep entries concise and actionable
- Use realistic income ranges (no hype)
- Link to reputable sources only
- Test skills before recommending
- Respect the [Awesome List Guidelines](https://github.com/sindresorhus/awesome/blob/main/contributing.md)

## Code of Conduct

Be respectful and constructive. We're all here to help each other make money with OpenClaw.
